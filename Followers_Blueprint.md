# Blueprint: Followers System

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **ระบบผู้ติดตาม (Followers System)** ซึ่งเป็นส่วนหนึ่งของ Socialive ใน MESY Universe.

## 1. แนวคิดหลัก (Core Concept)

- **Public Connections:** สร้างพื้นที่สำหรับการเชื่อมต่อแบบสาธารณะ ที่ผู้ใช้สามารถติดตามข่าวสาร, ผลงาน, และกิจกรรมของบุคคลที่น่าสนใจได้โดยไม่ต้องรอการอนุญาต.
- **One-way Relationship:** เน้นย้ำว่าการติดตามเป็นความสัมพันธ์แบบทางเดียว (One-way) ซึ่งแตกต่างจากการเป็นเพื่อน (Friends) ที่ต้องได้รับการยินยอมจากทั้งสองฝ่าย. เหมาะสำหรับ Creator, ศิลปิน, และบุคคลสาธารณะ.

## 2. ฟีเจอร์หลัก (Key Features)

- **Follow / Unfollow System:**
  - **การติดตาม:** ผู้ใช้สามารถกดปุ่ม "Follow" เพื่อติดตามผู้ใช้อื่นได้ทันที.
  - **การเลิกติดตาม:** ผู้ใช้สามารถกด "Unfollow" เพื่อยกเลิกการติดตามได้ทุกเมื่อ.

- **Follower & Following Lists:**
  - **Follower List:** หน้าสำหรับดูรายชื่อผู้ใช้ทั้งหมดที่กำลังติดตามเรา.
  - **Following List:** หน้าสำหรับดูรายชื่อผู้ใช้ทั้งหมดที่เรากำลังติดตามอยู่.

## 3. ความแตกต่างระหว่าง Friends และ Followers

- **Followers (ผู้ติดตาม):** เป็นความสัมพันธ์แบบทางเดียว (One-way) เหมาะสำหรับการติดตามบุคคลสาธารณะ, ศิลปิน, และ Creator เพื่อรับข่าวสาร.
- **Friends (เพื่อน):** เป็นความสัมพันธ์แบบสองทาง (Mutual) ทั้งสองฝ่ายต้องยอมรับซึ่งกันและกัน เหมาะสำหรับการเชื่อมต่อส่วนตัว.

## 4. หน้าและส่วนประกอบที่เกี่ยวข้อง (Key Pages & Components)

- **`/followers`:** หน้าหลักสำหรับจัดการทุกอย่างที่เกี่ยวกับผู้ติดตาม ซึ่งอาจรวมอยู่ในหน้าเดียวกับ `/friends` แต่แสดงคนละแท็บ (Tab).
- **Profile Page:** ปุ่ม "Follow" จะปรากฏบนหน้าโปรไฟล์ของผู้ใช้อื่น.
