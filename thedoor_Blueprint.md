# Blueprint: The Door (Member Login)

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับหน้า **The Door** ซึ่งเป็นประตูมิติสำหรับสมาชิก (Member) ในการเข้าสู่ Member Zone ของ MESY Universe ณ `/the-door`

## 1. แนวคิดหลัก (Core Concept)

- **Ceremonial Gateway:** เป็นมากกว่าแค่ฟอร์มล็อกอิน แต่เป็น "ประตูมิติ" ที่สร้างประสบการณ์อันศักดิ์สิทธิ์และพิเศษสำหรับสมาชิกในการเข้าสู่พื้นที่เฉพาะของตนเอง
- **Secure & Swift Access:** มอบวิธีการเข้าสู่ระบบที่ปลอดภัยและรวดเร็ว โดยใช้ข้อมูลเฉพาะของสมาชิก
- **Future-Ready Authentication:** ออกแบบโดยคำนึงถึงเทคโนโลยีการยืนยันตัวตนในอนาคต

## 2. องค์ประกอบหลัก (Key Components)

- **Gate Animation (อนิเมชันประตูมิติ):**
  - **รูปแบบ:** ประตูหินแกะสลักที่จะเปิดออกเมื่อผู้ใช้ล็อกอินสำเร็จ เผยให้เห็นอุโมงค์ดวงดาว (Vortex) ด้านใน
  - **เป้าหมาย:** สร้างความรู้สึกน่าตื่นเต้นและให้ความรู้สึกเหมือนกำลังเดินทางข้ามมิติ

- **Login Form (ฟอร์มล็อกอิน):**
  - **MESY Member ID:** ฟิลด์สำหรับกรอก ID เฉพาะที่สมาชิกได้รับหลังการลงทะเบียน
  - **PIN Code:** ฟิลด์สำหรับกรอกรหัส PIN 4 หลัก

- **Future Authentication Options (ตัวเลือกการยืนยันตัวตนในอนาคต):**
  - **Biometrics Icon:** ไอคอนรูปลายนิ้วมือ สำหรับการล็อกอินด้วยการสแกนนิ้ว/ใบหน้า (เตรียมไว้สำหรับอนาคต)
  - **QR Code Icon:** ไอคอนรูป QR Code สำหรับการล็อกอินผ่านแอปพลิเคชันมือถือ (เตรียมไว้สำหรับอนาคต)

- **Loading Vortex (อนิเมชันโหลด):**
  - **รูปแบบ:** หลังจากประตูกำลังเปิด จะแสดงเอฟเฟกต์อุโมงค์ดวงดาวที่กำลังเคลื่อนที่ เพื่อจำลองการเดินทางก่อนเข้าสู่หน้า Dashboard

- **Non-Member Guidance (คำแนะนำสำหรับผู้ที่ยังไม่ได้เป็นสมาชิก):**
  - แสดงขั้นตอนที่จำเป็น (ยืนยันอีเมล, ยืนยันเบอร์โทร, เชื่อมต่อการชำระเงิน, ชำระค่าสมาชิก) เพื่อเป็นแนวทางสำหรับผู้ใช้ทั่วไปที่ต้องการสมัคร

## 3. ขั้นตอนการทำงาน (Flow)

1.  ผู้ใช้เข้ามาที่หน้า `/the-door`
2.  ผู้ใช้กรอก **MESY Member ID** และ **PIN Code**
3.  กดปุ่ม "Unlock Gate"
4.  **สำเร็จ:**
    - ประตูมิติจะเริ่มเปิดออก
    - หน้าจอจะเปลี่ยนเป็นอนิเมชัน "Loading Vortex"
    - หลังจากอนิเมชันจบ, ผู้ใช้จะถูกนำทางไปยังหน้า Member Dashboard (`/dashboard`)
5.  **ไม่สำเร็จ:**
    - ระบบจะแสดงข้อความแจ้งเตือน (เช่น "Invalid Member ID or PIN")
    - ประตูจะยังคงปิดอยู่
