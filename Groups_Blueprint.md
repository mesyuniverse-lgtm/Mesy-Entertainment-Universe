# Blueprint: Groups System

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **ระบบกลุ่ม (Groups System)** ซึ่งเป็นส่วนหนึ่งของ Socialive ใน MESY Universe

## 1. แนวคิดหลัก (Core Concept)

- **Community Building:** สร้างพื้นที่สำหรับผู้ใช้ที่มีความสนใจในเรื่องเดียวกันได้รวมตัว, พูดคุย, และทำกิจกรรมร่วมกัน
- **Interest-Based Hubs:** เป็นศูนย์กลางของชุมชนย่อยๆ ที่ขับเคลื่อนด้วยความสนใจเฉพาะทาง ไม่ว่าจะเป็นเรื่องเกม, ศิลปะ, ดนตรี, หรือหัวข้ออื่นๆ

## 2. ฟีเจอร์หลัก (Key Features)

- **Group Creation:**
  - **การสร้างกลุ่ม:** สมาชิกสามารถสร้างกลุ่มของตัวเองได้
  - **ประเภทกลุ่ม:**
    - **Public Group (กลุ่มสาธารณะ):** ทุกคนสามารถมองเห็นและเข้าร่วมได้ทันที
    - **Private Group (กลุ่มส่วนตัว):** ต้องได้รับการอนุมัติจากผู้ดูแล (Admin) ก่อนจึงจะเข้าร่วมได้ เนื้อหาภายในจะเห็นได้เฉพาะสมาชิกเท่านั้น

- **Group Discovery:**
  - **ค้นหากลุ่ม:** ผู้ใช้สามารถค้นหากลุ่มจากชื่อหรือคีย์เวิร์ดได้
  - **แนะนำกลุ่ม:** ระบบจะแนะนำกลุ่มที่อาจตรงกับความสนใจของผู้ใช้

- **Group Feed/Wall:**
  - **การโพสต์:** สมาชิกในกลุ่มสามารถโพสต์ข้อความ, รูปภาพ, หรือวิดีโอได้ภายในกลุ่ม
  - **การโต้ตอบ:** สามารถกดไลค์, คอมเมนต์, และแชร์โพสต์ภายในกลุ่มได้

- **Member Management:**
  - **ผู้ดูแลกลุ่ม (Admin):** สามารถจัดการสมาชิก, ลบโพสต์ที่ไม่เหมาะสม, และแก้ไขข้อมูลกลุ่มได้

## 3. หน้าและส่วนประกอบที่เกี่ยวข้อง (Key Pages & Components)

- **`/groups`:** หน้าหลักสำหรับค้นหา, เข้าร่วม, และจัดการกลุ่มทั้งหมดที่ผู้ใช้เป็นสมาชิก
- **`/groups/[groupId]`:** หน้าของกลุ่มแต่ละกลุ่ม ซึ่งจะแสดงฟีด, รายชื่อสมาชิก, และข้อมูลของกลุ่ม
