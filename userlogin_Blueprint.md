# Blueprint: User Login Form

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **ฟอร์มการเข้าสู่ระบบสำหรับผู้ใช้ทั่วไป (User Login Form)** ของ MESY Universe ณ `/login`

## 1. แนวคิดหลัก (Core Concept)

- **Gateway for Returning Users:** เป็นประตูหลักสำหรับผู้ใช้ที่มีบัญชีอยู่แล้วในการกลับเข้าสู่จักรวาล MESY
- **Simple & Secure:** ออกแบบให้เรียบง่ายและปลอดภัยที่สุด โดยเน้นที่ฟังก์ชันการยืนยันตัวตนเป็นหลัก

## 2. องค์ประกอบและฟิลด์ในฟอร์ม (Form Fields & Components)

- **Email:** ฟิลด์สำหรับกรอกอีเมลที่ใช้ลงทะเบียน
- **Password:** ฟิลด์สำหรับกรอกรหัสผ่าน

## 3. ฟีเจอร์สนับสนุน (Supporting Features)

- **"Forgot Password?" Link:** ลิงก์สำหรับไปยังหน้าจอรีเซ็ตรหัสผ่าน
- **Social Login:** ตัวเลือกในการเข้าสู่ระบบผ่าน Google เพื่อความสะดวก
- **"Sign Up" Link:** ลิงก์สำหรับผู้ใช้ใหม่ที่ยังไม่มีบัญชีเพื่อไปยังหน้าสมัครสมาชิก

## 4. ขั้นตอนการทำงาน (Flow)

1.  **หน้าแรก (`/login`):** ผู้ใช้กรอก Email และ Password
2.  **การยืนยันตัวตน:** ระบบทำการตรวจสอบข้อมูลกับ Firebase Authentication
3.  **สำเร็จ:** หากข้อมูลถูกต้อง, ผู้ใช้จะถูกนำทางไปยังหน้า User Dashboard (`/users`)
4.  **ไม่สำเร็จ:** หากข้อมูลไม่ถูกต้อง, ระบบจะแสดงข้อความแจ้งเตือน (เช่น "รหัสผ่านไม่ถูกต้อง" หรือ "ไม่พบบัญชีนี้")
