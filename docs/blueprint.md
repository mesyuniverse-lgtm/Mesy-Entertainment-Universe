# **App Name**: Mesy Entertainment Universe

## Core Features:

- User Authentication: Secure user registration and login using email/password or Google OAuth. Implements a post-login popup to collect additional user information and prompt membership.
- Role-Based Dashboard: A dashboard that dynamically adapts based on user role (User, Member, Admin, Super-admin, Developer) displaying relevant information and actions. All roles can view the notifications center. User can view downline income with their corresponding levels in a structured and user-friendly table.
- AI-Powered Content Generation Demo: Demo of AI tools like content generator, avatar generator and playlist builder that will be available to all users. Generated results cannot be saved without logging in or signing up as a member.
- Downline and Level Management: Track user's downline members, levels, and calculate monthly income based on a tiered system with service fee deductions. Downline commission: level 0 to level 50 member table following assumption calculations that you gave me
- Admin and Role Management: Admin interface for managing members, roles, and system configurations. Super-admins will configure the UI/UX and be able to approve transactions. Admins will be able to suspend accounts and view stats.
- 3D Interactive Environment: Integrate Three.js for displaying interactive 3D environments, models, and animations based on a fantasy theme. Add parallax effects, glow effects, and micro-interactions to create an immersive UX, especially when users advance in the levels.
- AI Content Moderation: Utilize an AI tool to monitor user-generated content within public and private rooms for compliance with community standards. The AI tool can learn dynamically based on super-admin feedback.

## Style Guidelines:

- Primary color: Gold (#FFD700), reflecting the premium and fantasy aspects of the platform.
- Background color: Desaturated Gold (#E0D8B4), providing a soft and magical feel while keeping with the main hue.
- Accent color: Blue (#6495ED), used for interactive elements and highlights, providing a cool contrast.
- Headline font: 'Fantasy Magist' (titles) as requested by the user. Note: currently only Google Fonts are supported.
- Body font: 'Montserrat' (content) as requested by the user. Note: currently only Google Fonts are supported.
- Use 3D fantasy-themed icons, such as dragons, castles, and gems.
- Design a ceremonial journey-like UX with gamified progression and unlockable content, using progress bars and achievement badges.
- Incorporate parallax scrolling, hover glow effects, and micro-interactions to enhance user engagement.
