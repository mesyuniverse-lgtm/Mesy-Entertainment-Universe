# Blueprint: Welcome Page

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับหน้า **Welcome Page** ซึ่งเป็นหน้า Landing Page หลักของ MESY Universe ณ `/welcome`

## 1. แนวคิดหลัก (Core Concept)

- **First Impression:** เป็น "ประตูหน้า" ของ MESY Universe ที่ออกแบบมาเพื่อสร้างความประทับใจแรกและดึงดูดผู้ใช้ใหม่
- **Information Hub:** ทำหน้าที่เป็นศูนย์กลางข้อมูลสำหรับผู้ที่สนใจ เพื่อให้สามารถศึกษาและทำความเข้าใจภาพรวมทั้งหมดของแพลตฟอร์มได้ก่อนตัดสินใจเข้าร่วม
- **Conversion Funnel:** เป็นเส้นทางหลักในการนำทางผู้ใช้ทั่วไป (Guest) ไปสู่การสมัครสมาชิก (User) และการเป็นสมาชิกเต็มตัว (Member)

## 2. องค์ประกอบหลัก (Key Components)

- **Hero Section:**
  - **รูปแบบ:** Slideshow รูปภาพแฟนตาซีขนาดใหญ่ที่เล่นอัตโนมัติ เพื่อสร้างความตื่นตาตื่นใจ
  - **เนื้อหา:** หัวข้อที่ดึงดูด, คำโปรย, และปุ่ม Call-to-Action "Let's Startup" ที่ชัดเจน

- **Features Overview Section:**
  - **เป้าหมาย:** สรุปฟีเจอร์หลัก 3-4 อย่างของแพลตฟอร์ม
  - **รูปแบบ:** การ์ดแสดงข้อมูลพร้อมไอคอนและคำอธิบายสั้นๆ

- **Member Plan Teaser:**
  - **เป้าหมาย:** แนะนำระบบสมาชิกและรายได้เพื่อกระตุ้นความสนใจ
  - **รูปแบบ:** ตารางข้อมูลสรุประดับและรายได้เบื้องต้น พร้อมลิงก์ไปยังหน้ารายละเอียดเต็ม

- **Chronicle Teaser:**
  - **เป้าหมาย:** สร้างความโปร่งใสและน่าเชื่อถือโดยการแสดงแผนการพัฒนา
  - **รูปแบบ:** ส่วนสรุปสั้นๆ เกี่ยวกับ Roadmap ของโปรเจกต์ พร้อมลิงก์เพื่อดูรายละเอียดเพิ่มเติม

## 3. หน้าและส่วนประกอบที่เกี่ยวข้อง (Key Pages & Components)

- **`/welcome`:** หน้าหลักที่รวมส่วนประกอบทั้งหมดนี้ไว้
- **`/features`:** หน้าแสดงรายละเอียดฟีเจอร์ทั้งหมด
- **`/member-plan`:** หน้าแสดงรายละเอียดแผนสมาชิกและรายได้
- **`/chronicle`:** หน้าแสดง Roadmap ของโปรเจกต์
