# Blueprint: Universe Monitor

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **เครื่องมือสอดส่องจักรวาล (Universe Monitor)** ซึ่งเป็นส่วนหนึ่งของ Super Admin Zone ณ `/monitor`

## 1. แนวคิดหลัก (Core Concept)

- **Real-time Command Center:** เป็นศูนย์บัญชาการที่ช่วยให้ Super Admin สามารถสอดส่องกิจกรรม, พฤติกรรมของผู้ใช้, และสถานะของระบบทั้งหมดที่เกิดขึ้นในจักรวาล MESY ได้แบบเรียลไทม์
- **Observability & Intervention:** ไม่ใช่แค่การเฝ้าดู แต่ต้องสามารถ "แทรกแซง" และ "จัดการ" กับเหตุการณ์ที่เกิดขึ้นได้ทันที เพื่อรักษาความปลอดภัยและความเรียบร้อยของแพลตฟอร์ม
- **Data-Driven Oversight:** เป็นเครื่องมือที่แสดงข้อมูลเชิงลึก เพื่อช่วยในการตัดสินใจและแก้ไขปัญหาได้อย่างรวดเร็วและแม่นยำ

## 2. ฟีเจอร์หลัก (Key Features)

- **Real-time User Activity Feed:**
  - **เป้าหมาย:** แสดงกิจกรรมสำคัญของผู้ใช้ทั้งหมดแบบสดๆ
  - **ฟีเจอร์:**
    - ฟีดที่อัปเดตอัตโนมัติเมื่อมีกิจกรรม เช่น การสมัครสมาชิกใหม่, การเลื่อนระดับ, การทำธุรกรรมมูลค่าสูง, การสร้างเนื้อหาใหม่, การรายงานปัญหา
    - สามารถกรองกิจกรรมตามประเภทหรือโซนได้

- **User Management & Inspection:**
  - **เป้าหมาย:** เครื่องมือสำหรับจัดการผู้ใช้รายบุคคล
  - **ฟีเจอร์:**
    - ระบบค้นหาผู้ใช้จากชื่อ, อีเมล, หรือ ID
    - สามารถคลิกเพื่อดูข้อมูลเชิงลึกของผู้ใช้คนนั้นๆ ได้ (โปรไฟล์, ประวัติกิจกรรม, สถานะ, Role)
    - เครื่องมือสำหรับ Admin Actions เช่น การระงับบัญชี (Suspend), การเปลี่ยน Role, การรีเซ็ตรหัสผ่าน

- **Content Moderation Queue:**
  - **เป้าหมาย:** จัดการเนื้อหาที่อาจจะไม่เหมาะสม
  - **การทำงาน:** เป็นหน้าจอที่ดึงข้อมูลเนื้อหาที่ถูก AI ตรวจจับจากระบบ **AI Content Moderation** มาแสดงให้ Super Admin ตรวจสอบและตัดสินใจ (อนุมัติ, ลบ, หรือแบนผู้ใช้)

- **System Health Dashboard:**
  - **เป้าหมาย:** แสดงสถานะของระบบในเชิงเทคนิค
  - **ฟีเจอร์:**
    - กราฟแสดงการใช้งาน CPU, Memory, และ Database Load
    - สถานะการตอบสนองของ API และบริการภายนอก (เช่น AI models)
    - รายงานข้อผิดพลาด (Error Log) ล่าสุดของระบบ

- **Zone-Specific Monitoring:**
  - **เป้าหมาย:** สามารถเจาะลึกเพื่อดูกิจกรรมที่เกิดขึ้นในแต่ละโซนหลักได้
  - **ตัวอย่าง:**
    - **Socialive Monitor:** ดูกระแสการไลฟ์, จำนวนผู้ชม, ของขวัญ (Gifts) ที่ถูกส่ง
    - **Developer Zone Monitor:** ดู Quests ที่ถูกสร้าง, โปรเจกต์ที่มีคนทำเยอะที่สุด
    - **Shopping Hub Monitor:** ดูปริมาณการซื้อขาย, สินค้ายอดนิยม

## 3. หน้าและส่วนประกอบที่เกี่ยวข้อง (Key Pages & Components)

- **`/monitor`:** หน้าหลักของ Universe Monitor ซึ่งเป็นแดชบอร์ดที่รวมฟีเจอร์ทั้งหมดไว้ในหน้าเดียว
- **User Detail Modal/Page:** หน้าต่างหรือหน้าที่แสดงข้อมูลเชิงลึกของผู้ใช้แต่ละคนเมื่อถูกเลือกจากหน้า Monitor
