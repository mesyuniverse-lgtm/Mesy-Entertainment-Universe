# Blueprint: Live Streaming System

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **ระบบถ่ายทอดสด (Live Streaming System)** ซึ่งเป็นส่วนประกอบสำคัญทั้งใน Socialive และ Entertainment Hub ของ MESY Universe.

## 1. แนวคิดหลัก (Core Concept)

- **Real-time Interaction:** สร้างพื้นที่ให้ผู้ใช้ (VJ, ศิลปิน, สมาชิก) สามารถมีปฏิสัมพันธ์กับผู้ชมได้แบบเรียลไทม์ผ่านวิดีโอ, เสียง, และแชท
- **Creator Empowerment:** เป็นเครื่องมือสำหรับ Creator ในการสร้างฐานแฟนคลับ, แสดงความสามารถ, และสร้างรายได้โดยตรงจากผู้ชม
- **Community Gathering:** เป็นศูนย์กลางของการรวมตัวในกิจกรรมต่างๆ ตั้งแต่การพูดคุยสบายๆ ไปจนถึงคอนเสิร์ตขนาดใหญ่

## 2. ฟีเจอร์หลัก (Key Features)

- **Live Broadcasting:**
  - **เป้าหมาย:** ให้ Creator สามารถถ่ายทอดสดวิดีโอและเสียงของตนเองไปยังผู้ชมจำนวนมากได้
  - **ฟีเจอร์ย่อย:**
    - **Studio Mode:** หน้าสำหรับ VJ ในการตั้งค่าสตรีมก่อนเริ่ม เช่น ตั้งชื่อรายการ, เลือกฟิลเตอร์/เอฟเฟกต์, ทดสอบเสียงและภาพ
    - **Multi-Host:** รองรับการไลฟ์ร่วมกันหลายคนในหน้าจอเดียว

- **Real-time Chat:**
  - **เป้าหมาย:** เป็นช่องทางหลักให้ผู้ชมได้สื่อสารกับ VJ และผู้ชมคนอื่นๆ
  - **ฟีเจอร์ย่อย:**
    - **Public Chat:** ข้อความจะแสดงให้ทุกคนในห้องเห็น
    - **Admin/Mod Tools:** เครื่องมือสำหรับผู้ดูแลในการลบข้อความหรือแบนผู้ใช้ที่ไม่เหมาะสม

- **Virtual Gifts & Tipping:**
  - **เป้าหมาย:** เป็นช่องทางให้ผู้ชมสามารถสนับสนุน Creator ที่ชื่นชอบได้โดยตรง
  - **การทำงาน:** ผู้ชมสามารถใช้ MESY Coins เพื่อซื้อของขวัญ (Gifts) ซึ่งเป็นไอเท็มดิจิทัล (เช่น ดอกไม้, มงกุฎ, มังกร) และส่งให้ VJ ระหว่างการไลฟ์ได้ โดย VJ จะได้รับรายได้ส่วนหนึ่งจากมูลค่าของขวัญ

- **Ticketing System (for special events):**
  - **เป้าหมาย:** สำหรับการจัดการแสดงสดหรืออีเวนต์พิเศษที่ต้องมีการซื้อตั๋วเข้าชม
  - **การทำงาน:** ศิลปินหรือผู้จัดสามารถสร้างอีเวนต์, กำหนดราคาตั๋วเป็น MESY Coins, และจำกัดการเข้าชมเฉพาะผู้ที่มีตั๋วเท่านั้น

## 3. หน้าและส่วนประกอบที่เกี่ยวข้อง (Key Pages & Components)

- **`/live`:** หน้าหลักสำหรับค้นหาและดูรายการที่กำลังถ่ายทอดสดอยู่ (Live Now)
- **`/studio`:** หน้าสำหรับ Creator ในการตั้งค่าและเริ่มต้นการถ่ายทอดสดของตนเอง
- **Live Viewing Interface:** หน้าจอที่ผู้ใช้เห็นขณะรับชมการถ่ายทอดสด ซึ่งประกอบด้วย:
  - Video Player
  - Chat Box
  - Gift Sending Panel
  - VJ Profile Information
  - Viewer List
