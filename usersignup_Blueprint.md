# Blueprint: User Signup Form

เอกสารนี้คือพิมพ์เขียว (Blueprint) เฉพาะสำหรับ **ฟอร์มการสมัครสมาชิกสำหรับผู้ใช้ทั่วไป (User Signup Form)** ของ MESY Universe ณ `/signup`

## 1. แนวคิดหลัก (Core Concept)

- **First Step into the Universe:** เป็นประตูบานแรกสำหรับผู้ใช้ใหม่ในการเข้าสู่จักรวาล MESY
- **Profile Foundation:** เป็นขั้นตอนการเก็บข้อมูลที่จำเป็นเพื่อสร้างตัวตนดิจิทัล (Digital Identity) และโปรไฟล์เบื้องต้นของผู้ใช้
- **User-Friendly Experience:** ออกแบบฟอร์มให้ใช้งานง่าย, ไม่ซับซ้อน, และเป็นมิตรกับผู้ใช้

## 2. องค์ประกอบและฟิลด์ในฟอร์ม (Form Fields & Components)

ฟอร์มสมัครสมาชิกจะรวบรวมข้อมูลที่จำเป็นตาม `UserProfile` และ `User` entity ใน `backend.json`

- **ข้อมูลบัญชี (Account Info):**
  - **Email:** สำหรับการล็อกอินและการสื่อสารหลัก
  - **Password:** รหัสผ่าน (ต้องมีการยืนยันรหัสผ่าน)

- **ข้อมูลโปรไฟล์ (Profile Info):**
  - **Username:** ชื่อผู้ใช้ duy nhất ในระบบ (ต้องไม่ซ้ำกับคนอื่น)
  - **Firstname (ชื่อจริง):**
  - **Lastname (นามสกุล):**
  - **Nickname (ชื่อเล่น/ฉายา):**
  - **Mobile Number (เบอร์โทรศัพท์):**
  - **Date of Birth (วันเกิด):** แบ่งเป็น วัน, เดือน, ปี เพื่อให้เลือกง่าย
  - **Gender (เพศ):** ให้ผู้ใช้เลือก (ชาย, หญิง, หรือกำหนดเอง)

## 3. ขั้นตอนการสมัคร (Flow)

1.  **หน้าแรก (`/signup`):** ผู้ใช้กรอกข้อมูลทั้งหมดที่ระบุไว้ข้างต้น
2.  **การตรวจสอบ:** ระบบตรวจสอบข้อมูลเบื้องต้น เช่น อีเมลซ้ำ, รหัสผ่านไม่ตรงกัน
3.  **การสร้างบัญชี:**
    - เมื่อข้อมูลถูกต้อง, ระบบจะสร้าง `User` ใน Firebase Authentication
    - สร้างเอกสาร `User` และ `UserProfile` ใน Firestore พร้อมข้อมูลที่ผู้ใช้กรอก
4.  **การยืนยันอีเมล:** ระบบส่งอีเมลยืนยันไปยังผู้ใช้
5.  **เสร็จสิ้น:** นำผู้ใช้ไปยังหน้า User Dashboard (`/users`) พร้อมแสดงข้อความต้อนรับ

## 4. การเชื่อมต่อกับ Social Login

- **Google Login:** ผู้ใช้สามารถสมัครผ่าน Google ได้ หากเป็นผู้ใช้ใหม่, ระบบจะดึงข้อมูลพื้นฐาน (ชื่อ, อีเมล) มาจาก Google และอาจจะต้องให้ผู้ใช้กรอกข้อมูลเพิ่มเติม (เช่น Username) ในภายหลัง
